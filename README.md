# ReWheel
A fullstack website for building, saving and displaying customizable bikes rendered in 3D.  

**Database**
- MongoDB

**Backend**
- TypeScript
- NodeJS
- Apollo (apollo-server-express)
- GraphQL

**Frontend**
- Webpack
- React.js
- Three.js
- Redux
- TailwindCSS
