# wheelwhale

A fullstack website for building, saving and viewing customizable bikes rendered in 3D.

**Database**

- MongoDB

**Backend**

- TypeScript
- NodeJS
- Apollo (apollo-server-express)
- GraphQL
- Mongoose
- JWT

**Frontend**

- TypeScript
- Webpack
- React.js
- Apollo (apollo-client)
- Three.js
- Redux
- TailwindCSS
- Auth0
